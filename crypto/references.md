# References

#### 1. Crypto-attacks

{% embed url="https://github.com/jvdsn/crypto-attacks/" %}

A collection of practical cryptographic attack implementations covering a wide range of algorithms and vulnerabilities, including RSA, AES, ECC, and more. The repository provides ready-to-use scripts (though some may require modifications depending on the challenge) and test cases commonly encountered in Crypto CTFs.

#### 2. Recover seed

{% embed url="https://github.com/StackeredSAS/python-random-playground" %}

A Python's random module and the MT19937 PRNG. It explores seed recovery techniques, state reconstruction, and implementation details of Python's random number generation.

#### 3. Randcrack

{% embed url="https://github.com/tna0y/Python-random-module-cracker" %}

A Python library that reconstructs the internal state of Python's random module (MT19937) from 624 observed 32-bit outputs. Once the state is recovered, it can accurately predict all future random values generated by the same instance.

#### 4. Various MT19937 Seed Recovery References&#x20;

[https://gist.github.com/maple3142/1e3e81411a791f85073c3d902b0f14ef](https://gist.github.com/maple3142/1e3e81411a791f85073c3d902b0f14ef)

A collection of techniques and references for attacking the MT19937 (Mersenne Twister) pseudorandom number generator. It includes methods for recovering the generator's internal state, reconstructing its seed under different scenarios, and reversing the PRNG based on observed outputs.<br>
